# Geluiden

- **wait-music.wav** — Wachtmuziek tijdens het antwoorden. De app speelt telkens ~20 seconden uit een willekeurig stuk van het bestand, met fade-in en fade-out, zodat het steeds anders klinkt.
- **countdown.mp3** — Countdown “3, 2, 1, go” (4 seconden). Speelt na de voorgelezen vraag; daarna mag je antwoorden.
